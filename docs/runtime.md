
# Tachyon AI Runtime

## Why AI Runtime

There is MCP protocol and A2A protocol, both very similar, but server have to implement both separately using available SDKs.
_Tachyon AI Runtime_ serves as unified platform allowing multi-protocol communication and supporting more protocols in the future.

Example scenario where Task created with MCP protocol can be monitored via A2A, cancelled with REST, and have a subscription via Messaging/Kafka.
